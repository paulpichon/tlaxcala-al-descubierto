Esta es la versión 1.0.0 de esta API.

Información

Esta es una API solo de consumo : solo el método HTTP GET está disponible en los recursos. No se requiere autenticación para acceder a esta API, y todos los recursos están completamente abiertos y disponibles.

Política de uso justo

TAD es gratuito y está abierto para su uso. Debido a esto, le pedimos a cada desarrollador que cumpla con nuestra política de uso justo. A las personas que no cumplan con la política de uso justo se les prohibirá permanentemente su dirección IP.

TAD es principalmente una herramienta educativa, y no toleraremos ataques de denegación de servicio (DDoS) que impidan que las personas aprendan.

Normas:

  • - Almacenar (caché) localmente los recursos(de esta API) siempre que los solicite.
  • - Ser amable y amistoso con sus compañeros desarrolladores de TAD.

GET - Obtener historia(s) sobre Tlaxcala.

Llamar todos los registros de historias que hay sobre Tlaxcala, cabe destacar que el tipo de historia puede ser "RESUMEN" ó "EXTENDIDA".

Por defecto traera todos los registros, no hay paginación, ya que se considera tener pocos registros de historias.

Endpoint

GET https://tad-backend-1dkj-dev.fl0.io/api/historia-tlaxcala

Response:

	
	{
		"historias": [
			{
			"_id": "64c848e313bbef87435e6eb3",
			"titulo": "Tlaxcala a traves de la historia (Resumen)",
			"texto": "La historia de Tlaxcala...",
			"tipo_historia": "RESUMEN",
			"estado_visible": true
			}
		]
	}
Descripción
Nombre Descripción Tipo
_id ID de la historia. ObjectId
titulo Titulo de la historia. String
texto Texto de la historia (Puede ser un resumen o una historia extendida). String
tipo_historia Se dice el tipo de historia (RESUMEN o EXTENDIDA). String
estado_visible El estado de la historia sera TRUE o FALSE, SOLO las historias en TRUE se mostraran. Boolean
GET - Obtener señoríos Tlaxcaltecas.

Un señorío tlaxcalteca se refiere a una entidad política y territorial en la región que hoy conocemos como Tlaxcala, que existió antes de la llegada de los españoles a América en el siglo XVI, se mostrarán los 4 señoríos por defecto. Si lo que buscas es un solo señorío podrás usar el siguiente ENDPOINT para llamarlos por medio de su ID.

Endpoint

GET https://tad-backend-1dkj-dev.fl0.io/api/senorios

Response:

	
	{
		"senorios": [
			{
				"_id": "64e803a232a86fa0bff63f17",
				"nombre": "Tepeticpac",
				"imagen": "https://res.cloudinary.com/dy9prn3ue/image/upload/v1691808118/Tepeticpac_xdmoyx.jpg",

				"historia_antigua": "Según las crónicas históricas...",
				"economia_comercio": "La economía de...",
				"participacion_conflictos": "Los tlaxcaltecas, incluyendo el señorío de Tepeticpac...",
				"alianza_españoles": "La alianza entre los tlaxcaltecas...",
				"epoca_colonial": "Durante la época colonial en Tepeticpac..."
			}
			.
			.
			.
		]
	}
Descripción
Nombre Descripción Tipo
_id ID del señorío. ObjectId
nombre Nombre. String
imagen Jeroglífico del señorío. String
ubicacion Región donde se localizaba. String
historia_antigua Un poco de historia. String
economia_comercio Principales rasgos de su economía. String

alianza_españoles Resumen sobre la alianza con los españoles. String
epoca_colonial El señorio durante la epoca colonial. String

Se muestran todos los guerreros tlaxcaltecas

GET - Obtener haciendas tlaxcaltecas.

Las haciendas tlaxcaltecas, al igual que las haciendas en otras partes de México, jugaron un papel importante en la historia de la región. Estas haciendas eran grandes propiedades rurales que se establecieron durante la época colonial española y que estaban dedicadas principalmente a la agricultura y la ganadería.

Hoy en día, algunas de las haciendas en Tlaxcala se han convertido en atracciones turísticas, hoteles o centros culturales que muestran la historia y la arquitectura colonial de la región. También son un recordatorio importante de la historia agraria y social de México durante la época colonial y postindependencia.

Endpoint

GET https://tad-backend-1dkj-dev.fl0.io/api/senorios

Response:

	
	{
		"haciendas": [

				"_id": "651e09beb4e7ff703d79c747",

				"historia": "Una hacienda pulquera...",
				"ubicacion": "Esta hacienda se...",
				"atractivos": "El lugar proporciona...",
				"alojamiento": true,
				"eventos": true,
				"restaurante": false,
				"img": "[\"https://res.cloudinary.com/dy9prn..."
			},

			.
			.
			.
	}
Descripción
Nombre Descripción Tipo
_id ID de la hacienda. ObjectId
nombre_hacienda Nombre. String
historia Historia de la hacienda. String
ubicacion Ubicacion. String
atractivos Atractivos turisticos. String
alojamiento Alojamiento en la hacienda. Boolean
eventos Sí se pueden realizar eventos en el lugar. Boolean
restaurante Restaurante en el lugar. Boolean
img Imagenes de la hacienda. String
