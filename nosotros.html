Sobre Nosotros

¿Qué es Tlaxcala Al Descubierto (TAD)?

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permiten la comunicación entre diferentes aplicaciones de software. Define cómo las aplicaciones deben interactuar entre sí, especificando los métodos y formatos de datos que se utilizan para enviar solicitudes y recibir respuestas.

TAD, habla sobre la creación de una API sobre la historia de Tlaxcala, que al mismo tiempo ofrece la oportunidad de compartir el valioso legado cultural de la región a través de la tecnología.

Esta API brinda acceso a datos históricos confiables y recursos relacionados, permitiendo a los desarrolladores crear aplicaciones, sitios web y juegos educativos que divulguen la historia de Tlaxcala.

Con una interfaz documentada y servicios como consultas de eventos históricos y ubicaciones de sitios históricos, los desarrolladores pueden aprovechar la API para potenciar su creatividad y habilidades técnicas.

Además, esta API promovería la colaboración entre desarrolladores, permitiendo contribuciones y extensiones que enriquezcan su funcionalidad.

En resumen, la API de historia de Tlaxcala impulsa la innovación tecnológica y la difusión de la historia, ofreciendo a los desarrolladores una plataforma para poner en práctica sus habilidades y compartir el legado histórico de Tlaxcala.

Entonces, ¿quién construyó esto?

TAD V1 (Versión 1) fue creado por Paúl Pichón como un proyecto de fin de semana, pero rápidamente se convirtió en más que el trabajo de un fin de semana. En junio de 2023, Paúl desaprobó la V1 (Versión 1) a favor de trabajar en V2 (Versión 2).

Esteramos que intervengan más desarrolladores a este gran proyecto para que esta API siga creciendo.

¿De dónde se sacan todos estos datos?

Recopilamos la información de este sitio de varios recursos:

  • La mayoría de información viene de fuentes que se consideran confiables en Internet.
  • Un gran apoyo ha sido ChatGPT, una tecnología increíble que nos ha ayudado a reducir los tiempos de búsqueda de información, pero aun así nos tomamos el tiempo de verificar dicha información.
  • La imágenes vienen de redes sociales como Instagram y Facebook.
  • ¡Con el tiempo esperamos que más personas se unan a este proyecto, necesitamos más información y muchas, muchas imágenes!

Stack Tecnológico

El stack tecnológico está conformado hasta el día de hoy en el FRONTEND con HTML5, CSS3 y VANILLA JS.

En la parte del BACKEND con NODE JS para crear una API RESTful y combinarlo con una base de datos MONGODB.
